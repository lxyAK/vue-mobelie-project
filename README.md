# vue-mobile-project
这个一个简单的移动端SPA应用。包含图片浏览，实时评论，商品购物。后台使用Node+Express搭建的
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
